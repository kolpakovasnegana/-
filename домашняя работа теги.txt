    <p class="pk"; style="text-align:center;">Текстовые теги HTML</p>
    <p>1.тег "p" создт новый параграф. пример:</p>
    <p>Первый пораграф</p>
    <p>Второй параграф</p>

    <p>2.тег "b" выделяет текст жырным шрифтом :</p>
    <b>Пример выделения текста</b>

    <P>3.тег "hr" добавляет горезантальную линию. :</P>
    <p> Вот так выглядит горизантальная черта
    <hr>
    <P>4.тег "strong" выделяет жырным шрифтом ключевые слова. :</P>
    <p>эта страница написана на <strong>HTML</strong></p>

    <p>5.тег "br/" переводит текст на новую строку. :</p>
    <p>например мы используем очень длинный текст<br/>который надо поделить на несколько строк</p>

    <p>6.тег "nobr" запрещает перенос строки. :</p>
    <nobr>Например очень длинный текст который не надо переносить на две или более строк</nobr>

    <p>7.тег "sub" делает подиндекс. :</p>
    H<sub>2</sub>0

    <p>8.тег "sup" делает надиндекс. :</p>
    X<sup>2</sup>=4

    <p>9.тег "big" выводит более крупным чем остальной текст (тег устарел). </p>
    <big>Пример</big>

    <p>10.тег "small" делает текст более мелким чем остальной. </p>
    <small>Пример</small>

    <p>11.тег "i" выделяет текст курсивом.</p>
    <i>Пример</i>

    <p>12.тег "em" выделяет текст курсивом. </p>
    <em>Пример</em>

    <p>13.тег "s" делает текст зачеркнутым. </p>
    <s>Пример</s>

    <p>14.тег "tt" выделяет текст моноширинным шрифтом (тег устарел). </p>
    <tt>Пример</tt>

    <p>15.тег "kbd" выделяе текст моноширинным шрифтом. </p>
    <kbd>Пример</kbd>

    <p>16.тег "u" делает текст подчеркнутым. </p>
    <u>Пример</u>

    <p>17.тег "pre" предворительно отформатированный текст. </p>
    <pre>Пример</pre>

    <p>18.тег "marquee" заставляет текст перемещаться из стороны в сторону. </p>
    <marquee>Пример</marquee>
    
    <p>19.тег "q" включает в документ короткую цытату. </p>
    <q>Этот текст является цитатой</q>

    <p>20.тег "blockquote" преднозначен для включение в текст длинной цитаты. пример:</p>
    <blockquote>этот текст является достаточно длинной цитатой для этой страницы</blockquote>

    <p>21.тег "address" применяют для указаний сведений об авторе. пример:</p>
    <address>Мои сведения</address>

    <P>22.тег "cite" используется для цитат пример</P>
    <cite>этот текст цитата</cite>

    <P>23.тег "article" определяет независимый, самодостаточный контент, он поддерживается HTML 5. пример:</P>
    <article>
        <h1>Зачем нужен тег "article"?</h1>
        <p>Он нужен для определения независимого, самодостаточного контента</p>
    </article>

    <p>24.тег "section" нужен для определения размера в документе, он поддерживается HTML 5. пример:</p>
    <section>
        <h1>тег "Section"</h1>
        <p>тег для HTML</p>
    </section>

    <p>25.тег "header" нужен для определения шапки документа или страницы, он поддерживается HTML 5. пример:</p>
    <header>
        <h1>как использовать тег "header"</h1>
        <p>Использовать тег "header" нужно как заголовок</p>
    </header>

    <p>26.тег "footer" используется для определения (подвала) документа или раздела, он поддерживается HTML 5. пример:</p>
    <footer>
        <p>мои инициалы</p>
        <p>контактная информация</p>
    </footer>

    <p>27.тег "nav" определяет набор ссылок навигации (он преднозначен для основного блока информации), он поддерживается HTML 5. пример:</p>
    <nav>
        <a href='#'>Сылка</a> |
        <a href='#'>Сылка</a> |
        <a href='#'>Сылка</a> |
        <a href='#'>Сылка</a>
        </ul>
      </nav>
    
    <p>28.тег "aside" определяет некий контент,  находящийся в стороне от контента, внутри которого он расположен (как боковой блок страницы, "сайдбар"),он поддерживается HTML 5. пример: </p>
    <aside>
        <h4>что такое "aside"?</h4>
        <p>"aside" это тег</p>
    </aside>

    <p>29.тег "figure" и "figcaption" добавляют картинку, и описание её (поддерживаются HTML 5). пример:</p>
    <figure>
        <img src=c:\Users\Пользователь\Pictures\132131313.png alt="jpg файл" width="304" height="228">
        <figcaption>Рис. 1 — jpg картинка</figcaption>
     </figure>

    <p>30.тег "a" нужен для создания гиберссылок. пример:</p>
    <a href="сюда вставлять гиперссылку">ссылка</a>

    <P>31.тег "ul" Маркированный список. Каждый элемент списка оборачивается тегами "li". пример:</P>
    <ul>
       <li>Текст</li>
       <li>Текст</li>
       <li>Текст</li>
    </ul>

    <p>32.тег "ol" Нумерует список. Каждый элемент списка оборачивается тегами "li". пример:</p>
    <ol>
       <li>Текст</li>
       <li>Текст</li>
       <li>Текст</li>
    </ol>

     <p>33.тег "dl" список терминов. Внутри используются теги: "dt" -название термина, "dd" - описание термина. пример:</p>
     <dl>
        <dt>Заголовок 1</dt>
        <dd>текст 1</dd>
        <dt>Заголовок 2</dt>
        <dd>текст 2</dd>
        <dt>Заголовок 3</dt>
        <dd>текст 3</dd>
     </dl>

     <p>34.тег "bdo" задает направление текста (с помощью атрибута). пример:</p>
     <bdo dir="rtl">
        этот текст был перевернут
     </bdo>
